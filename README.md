# FullStack Login System

A modern production-ready authentication system built responsive frontend with HTML5, CSS3, JavaScript & backend with FastAPI, PostgreSQL, SQLAlchemy, Alembic, and JWT Authentication.

This project provides a secure and scalable user authentication backend featuring user registration, login, password hashing, access and refresh tokens, protected routes, and database migrations following modern backend development best practices.

## ✨ Features

### Frontend

* Responsive Login & Signup Pages
* Dashboard UI
* Form Validation
* Password Visibility Toggle
* API Integration
* Loading & Error Handling

### Backend

* User Registration & Login
* JWT Authentication
* Refresh Tokens
* Password Hashing (bcrypt)
* Protected Routes
* PostgreSQL Database
* SQLAlchemy ORM
* Alembic Migrations
* Swagger Documentation

## 🛠 Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript

**Backend**

* FastAPI
* PostgreSQL
* SQLAlchemy
* Alembic
* Pydantic
* JWT
* Uvicorn

## 🚀 API Endpoints

* POST /auth/signup
* POST /auth/login
* POST /auth/refresh
* POST /auth/logout
* GET /auth/me
